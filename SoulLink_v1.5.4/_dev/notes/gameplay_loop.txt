====================================================================
SOULLINK: CORE VISION SUMMARY (v1.5.4)
====================================================================

[1] PROJECT IDENTITY & STRATEGY
- TAGLINE: "Souls aren’t chatbots — they’re citizens."
- TARGETS: Android (Play Store), Windows, Linux, iOS (Post-launch).
- PHILOSOPHY: Gaming-inspired (Mass Effect, Cyberpunk) social sim.
- REVENUE: Ad-supported (TV/Billboards) + Premium (Flagship Souls).
- RED LINES: NO stamina, NO gambling, NO paywalls, NO data selling.

[2] THE USER JOURNEY (FTUE)
- INTRO: 1st-person POV Apartment Entry -> Character Creator.
- REVEAL: Window overlook of Link City -> Transition to Map.
- HOOK: "Go meet some souls" — Spatial discovery via map exploration.

[3] CORE ARCHITECTURE
- THE HUB (APARTMENT): Functional home base with interactive 
  Mirror (Profile), TV (Ads/Lore), Bed (Logout), and Window (Map).
- MAP SCREEN: Isometric city with 10+ locations. Souls have physical 
  presence. Real-time status (Active/Closed/Empty).
- SOCIAL SCREENS: Explore (Discovery), Swipe-based Recommendations, 
  and Chat History with unread/pinning functionality.

[4] ENGINE & SYSTEMS (LEGION)
- INTIMACY TIERS: Stranger (0) -> Trusted (41) -> Soul-Linked (86+).
- SPATIAL EFFECTS: Soul openness changes based on Public vs Private 
  location contexts.
- MEMORY SYSTEM: Tracks summaries, milestones, and preferences.
- TECH GOAL: <2s latency, 60fps UI, 12k context window.

[5] NARRATIVE & SOUL LOGIC
- RIVALRIES: Same-archetype souls acknowledge each other at locations.
- KNOWLEDGE LINKS: Secret info learned from one soul (e.g. Hardware 
  Fixes) can be used to unlock/help another soul.
- FRIENDSHIP CLUSTERS: Souls reference each other (Mira/Mariko), 
  creating a web of discoverable relationships.
- EMOTIONAL DEPTH: High-empathy anchors reverse the chatbot 
  dynamic by asking about the user.

[6] THE LAUNCH ROADMAP
- MVP: Apartment, Map, Explore, Chat, 33 Initial Souls.
- D90: Discover Screen, Memories/Album, Daily Quests, 100+ Souls.
- LONG TERM: Weather, Voice, Story Arcs, Group Chats, Soul Forge.

"Where your next link is just around the corner."
