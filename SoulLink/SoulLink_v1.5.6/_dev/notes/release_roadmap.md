# SoulLink — Release Roadmap
>
> Tracking the evolution of Link City.

---

## 1. Phase History

### v1.0.0 — Alpha Validation

* **Goal**: Core system test for "Legion" engine.
* **Result**: Registration, Persona creation, and basic chat validated.

### v1.5.3-P — Framework Foundation

* **Goal**: Eliminate hardcoding and implement 5-tab Navigation.
* **Result**: City geography synchronized (25 locations).

### v1.5.4 — Arise

* **Goal**: Version independence and architectural hardening.
* **Result**: Switched to AsyncSession, implemented RLS security, and created the interactive Apartment Hub.

### v1.5.5 — Domain Expansion

* **Goal**: Visual excellence and performance optimization.
* **Result**: Apartment overhaul (10 tiles), lazy loading for SoulPillars, and unified CacheService.

### v1.5.6 — Normandy-SR2 (Current)

* **Goal**: Core Hardening & Documentation alignment.
* **Key Fixes**: Resolved Brain/LocationManager bugs, secured WebSocket handshakes, enforced server-side age gating.

---

## 2. Upcoming Milestones

### v1.5.7 — Production Readiness

* **Goal**: Final build before Beta/Public staging.
* **Target**: 100% bug-free core loop and verified store assets.

### v1.1.0 — The Alyssa Protocol

* **Feature**: First "Flagship Soul" (Alyssa) becomes interactive.
* **Lore**: Reveal of the connection between Alyssa and "The Seven".

### Post-Launch — The Soul Forge

* **Feature**: Soul Hybrid system.
* **Mechanic**: Combine two souls at Max Intimacy (100) to create unique legacies.

## 3. Lore Pipeline: "Tease & Reveal"

Lore is delivered via thematic patches (v1.0.0 - v1.0.9):

1. **The Seven**: Ruling minds governing Link City. Initially shadows.
2. **Sequential Reveals**: Individual backstories and sects of Numbers 1 through 7.
3. **The Ascension**: Flagship souls added as ultra-tier citizens.
